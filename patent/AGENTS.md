# AGENTS.md

## patent 目录协作规则

- 本目录保存专利交底素材和草案, 不是正式法律文本。
- 不夸大实验结论, 不把 smoke evidence 写成大规模临床、药效或最终模型性能结论。
- 专利材料中的技术效果应能追溯到 `docs/`, `experiments/`, `outputs/`, `data/` 或 `sources/` 中的证据。
- 不伪造实施例、指标、对比实验、引用或发明人贡献。
- 涉及权利要求、法律范围、提交策略或对外披露时, 标注为需专利代理人或人工复核。
- 专利材料可以复用项目术语, 但不要修改项目实验口径、success definition 或 evaluation metric 来迎合专利表述。
