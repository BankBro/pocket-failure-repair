# 专利材料目录

本目录用于沉淀 `pocket-failure-repair` 项目中可供后续撰写专利交底书的技术材料。

## 文件说明

- `technical_disclosure_outline.md`: 当前课题的专利交底素材草案, 包括技术领域、背景问题、技术方案、创新点、实施流程、实验支撑和权利要求方向。
- `cn_patent_draft.md`: 中文专利草案初稿。
- `cn_patent_draft_revised.md`: 中文专利草案修订稿。
- `patent_agent_memo.md`: 面向专利代理人或人工复核的说明 memo。

## 当前课题一句话

面向蛋白口袋约束 3D 局部分子编辑, 给定一个已经失败的局部编辑候选分子, 利用可解释失败诊断反馈, 在相同 editable region 内生成修复后的候选分子, 并通过 repair/recovery gain 评价修复效果。

## 使用注意

- 本目录材料是技术交底准备稿, 不是正式专利文本。
- 后续撰写专利时应由专利代理人进一步处理权利要求、实施例和法律范围。
- 当前实验结果主要是机制级 smoke evidence, 不应在专利或论文中夸大为大规模临床、药效或最终模型性能结论。
- 技术效果、实施例和对比结论应能追溯到 `docs/`, `experiments/`, `outputs/`, `data/` 或 `sources/` 中的证据。
